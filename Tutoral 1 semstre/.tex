\section{Resumen}
\begin{frame}{Resumen}
\justifying
Se presentan simulaciones de trayectorias de brincos cuánticos, diseñadas para analizar el fenómeno de la fluorescencia resonante intermitente. Este análisis se centra en la estadística de los tiempos de espera, así como en las distribuciones de períodos brillantes y oscuros, como un paso inicial en el estudio de sistemas capaces de generar espectros con picos angostos.
\\
Además, se incluye una simulación de detección heterodina en un átomo de dos niveles.

\end{frame}




\section{Introducción}
\begin{frame}{Introducción}
\justifying
Contexto general:
\begin{itemize}
    \item La espectroscopía de fluorescencia es clave para estudiar sistemas cuánticos.
    \item Fenómenos como la fluorescencia intermitente revelan detalles de la dinámica de átomos individuales.
    \item La detección heterodina y homodina permiten espectros de alta resolución.
\end{itemize}
Teoría base: 
\begin{itemize}
    \item La teoría de trayectorias cuánticas modela eventos estocásticos como emisiones de fotones.
    \item Se emplea para simular sistemas abiertos y analizar períodos brillantes y oscuros.
\end{itemize}
\end{frame}


\section{Objetivos}
\begin{frame}{Objetivos}
Objetivo General:
\begin{itemize}
    \item Simular sistemas cuánticos de dos y tres niveles utilizando la teoría de Trayectorias Cuánticas (Función de onda de Monte Carlo).
\end{itemize}

Objetivos Específicos:
\begin{itemize}
    \item Simular un sistema de dos niveles, aplicando el modelo de trayectorias cuánticas para
explorar su comportamiento estocástico.
    \item Extender la simulación un sistema de tres niveles con intermitencia, analizando los efectos de este fenómeno en la dinámica del sistema.

    \item Obtener distribuciones de tiempos de espera y tiempos oscuros para ambos sistemas,
utilizando la ecuación maestra y trayectorias de brincos cuánticos.
    \item Simular la detección heterodina en un sistema de dos niveles, evaluando su capacidad
para obtener espectros de alta resolución.
\end{itemize}





\end{frame}

\begin{frame}{Estadística de tiempos oscuros}
\justifying
El procedimiento de c\'alculo de trayectorias permite recoger la informaci\'on de fotoemisiones y estad\'istica de los per\'iodos brillantes y oscuros. Esto se logra mejor si hacemos una trayectoria muy larga en lugar de muchas trayectorias cortas. Se pueden obtener las distribuciones de las duraciones de los periodos oscuros y brillantes, que siguen distribuciones exponenciales:\footnote{\fullcite{PlKn}}
$$ P_O = \frac{1}{T_O} e^{-t/T_O}, \qquad  
    P_B = \frac{1}{T_B} e^{-t/T_B} ,$$
con duraciones promedio
$$ T_O = \gamma_a^{-1}, \qquad T_B = \frac{2\Omega^2 + \gamma^2 + 4\Delta^2}{\gamma_d \Omega^2} .$$
\end{frame}

    \begin{itemize}

\begin{frame}{Conclusión}
\justifying
Se ha demostrado el poder de la teoría de trayectorias cuánticas para modelar y analizar fenómenos de fluorescencia resonante en sistemas de dos y tres niveles. Las simulaciones realizadas han permitido:
\begin{enumerate}
  

  \item Comprender el Comportamiento Estocástico.

  \item Validar Modelos Teóricos.

  \item Explorar Dinámicas Complejas en el sistema de tres niveles.
\end{enumerate}
Finalmente, este trabajo sienta las bases para futuras investigaciones sobre espectros de alta resolución y otros fenómenos cuánticos avanzados, mostrando la relevancia de las trayectorias cuánticas para el estudio de sistemas abiertos.

\end{frame}

    \begin{frame}{Algoritmo de Saltos Cuánticos}
        \justifying
        \vspace{0.3cm}
    
        \begin{enumerate}
            \item \textbf{Inicialización}: El sistema comienza en un estado puro inicial $\ket{\psi(0)}$.
            \item \textbf{Evolución Continua}: Se resuelve la ecuación de Schrödinger no hermítica:
            \[
            \frac{d}{dt}\ket{\psi(t)} = -\frac{i}{\hbar} H_{\text{eff}} \ket{\psi(t)}
            \]
            donde $H_{\text{eff}} = H - \frac{i}{2} \sum_j \gamma_j C_j^\dagger C_j$.
            
            \item \textbf{Detección de Saltos}: La probabilidad de salto en cada $\delta t$ se compara con un número aleatorio.
            
            \item \textbf{Salto Cuántico}: Si ocurre un salto, el estado colapsa a través de un operador $C_j$, y se normaliza:
            \[
            \ket{\psi(t)} \rightarrow \frac{C_j \ket{\psi(t)}}{\sqrt{\langle \psi(t) | C_j^\dagger C_j | \psi(t) \rangle}}.
            \]
            
            \item \textbf{Repetición}: Se repiten los pasos 2-4 hasta completar el tiempo de evolución.
        \end{enumerate}
    
    
    
        \vspace{0.2cm}
        \footnotesize
        \footfullcite{Carmichael}
    \end{frame}

    \begin{frame}{Algoritmo para Calcular Períodos Oscuros}
    
    \textbf{Descripción Simplificada del Algoritmo}
    
    \begin{enumerate} \item \textbf{Registro de Eventos de Colapso} \begin{itemize} \item Durante la simulación, se registran eventos de colapso en ciertos tiempos. \item Cada colapso corresponde a una transición específica en el sistema. \end{itemize} \vspace{0.5em} \item \textbf{Identificación de Períodos} \begin{itemize} \item \textbf{Períodos Brillantes}: Intervalos donde el sistema emite fluorescencia. \item \textbf{Períodos Oscuros}: Intervalos donde no hay emisión de fluorescencia. \end{itemize} \vspace{0.5em} \item \textbf{Detección de Transiciones Clave} \begin{itemize} \item Se enfocan en colapsos específicos que marcan el inicio y fin de los períodos. \item Por ejemplo, un colapso de tipo 1 indica el inicio de un período brillante. \end{itemize} \vspace{0.5em} \item \textbf{Cálculo de Duraciones} \begin{itemize} \item \textbf{Períodos Brillantes}: Duración = tiempo de fin - tiempo de inicio. \item \textbf{Períodos Oscuros}: Duración = inicio del siguiente período brillante - fin del período brillante actual. \end{itemize} \vspace{0.5em} \item \textbf{Almacenamiento y Análisis} \begin{itemize} \item Se guardan las duraciones para análisis estadístico. \item Permite calcular tiempos promedio y visualizar patrones de intermitencia. \end{itemize} \end{enumerate}
    
\end{frame}
